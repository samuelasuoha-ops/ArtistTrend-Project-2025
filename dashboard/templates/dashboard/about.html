About ArtistTrend

What is ArtistTrend?

ArtistTrend is a music analytics web application that allows users to search, compare, and monitor the popularity of music artists using live data from the Spotify Web API. It is designed to provide an easy visual understanding of how artists trend over time.

Key Features

  • Search for artists and view popularity, followers, genres, and images
  • Trending artists displayed based on real data and user search behavior
  • Compare two artists and view a chart showing popularity vs follower count
  • Artist detail pages including profile information and statistics
  • Spotify-powered insights updated automatically or manually through scheduled tasks

Technologies Used

  • Django - Web framework powering the application
  • Python - Backend processing and API integration
  • Spotify Web API - Live artist metadata and statistics
  • Chart.js - Interactive data visualization
  • SQLite - Persistent storage for artist data
  • HTML, CSS, JavaScript - Frontend structure and styling
